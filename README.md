# weather-forecast 天气预报
基于java swing的天气预报查询软件，使用[和风天气的API](https://www.heweather.com/documents/api/s6)
### 软件配置
1. Java环境：JDK1.8 + Eclipse
2. 数据库：MySql 5.7 + Navicat Premium 12
### 使用步骤
1. 将文件中的city.sql导入到你自己的数据库
2. 项目所需的jar包都在jar文件中，可自己在eclipse中配置进工程
3. 修改关于数据库连接的代码，改成你自己的数据库名，用户名及密码
![代码修改1](https://github.com/w-Bro/weather-forecast/blob/master/image/img_2.jpg)
4. 修改关于API连接的用户key，改成你在和风天气申请的用户key
![代码修改2](https://github.com/w-Bro/weather-forecast/blob/master/image/img_3.jpg)
